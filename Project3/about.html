About

The REST Countries API is a comprehensive and widely-used web service that provides detailed information about countries around the world. It offers a wide range of data, including country names, regions, subregions, populations, capital cities, area sizes, languages, currencies, and more. I decided to build this project using the REST Countries API, because of its simple and easy-to-use interface, and well documented documentation. Hope you enjoy!
